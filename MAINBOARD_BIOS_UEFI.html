Mainboard

BIOS

  • Englisch - Basic Input Output System
  • von IBM entwickelt
  • ist im System-ROM auf dem Mainboard gespeichert
  • ist die erste Software komponente die nach dem starten des PCs ausgeführt wird
  • lädt zuerst Software in den Hautspeicher (meist ein Betriebs System OS)
  • ersetzt somit das manuelle eingeben oder auswählen von zu ladener Software
  • unterstützt Treibersoftware für Hardware, wie z.B.: Tastatur, speicher systeme wie USB-Sticks und anderem
  • einheitliche Treibersoftware für mehrere Prozessoren und Betriebssystemen
  • wird aber oft nicht mehr von Betriebs Systemen benutzt, weil es nur die nötigsten Funktionen bietet und für ältere Systeme entwickelt wurde

UEFI

  • Englisch - Unifyed Extnesible Firmware Interface
  • von Intel entwickelt
  • urspünglich für den Intel Itanium Prozessor als EFI (Extensible Firmware Interface) entwickelt
  • Nachfolger des BIOS und deswegen auch oft noch als UEFI-BIOS oder BIOS-Setup bezeichnet
  • unterstützt im gegensatz zum BIOS:
    • GUID-Partitionstabelle ersetzt den Master Boot Record vom BIOS ist aber noch Kompatibel mit ihm
    • Grafikunterstützung für Framebuffer
    • Netzwerkfunktionen wie das booten über ein Netzwerk
    • seit UEFI 2.3.1 Secure Boot welches nur das Starten von signierter Software ermöglicht und so das Starten von Schardsoftware verhindern soll
  • das UEFI ist unaghängig vom CPU typ und unterstützt unterschiedliche Systeme und Architekturen, wie ARM
  • unterstützt Festplatengrößen von über 2 TB
  • ist erweiterbar
  • Daten werden in einer eigenen Festplatten Partition gespeichert
  • ermöglicht das permanentes speichern von Daten, wie boot Reienfolge, im NVRAM (Non-Volatile RAM) eine RAM Art die nach dem Ausschalten des Computers keine Daten verliert
